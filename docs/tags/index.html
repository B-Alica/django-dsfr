Design System de l‘État

  • Menu latéral (sidemenu)

  • Mise en exergue (highlight)

  • Pagination (pagination)
  • Sommaire (summary)
  • Tableau (table)
  • Tuile (tile)

Tags supplémentaires

  • CSS global
  • Groupe d’accordéons (accordion_group)
  • JS global

En-tête et pied de page

Les composants « En-tête (header) » et « Pied de page (footer) » sont actuellement gérés avec des balises include ( {% include "dsfr/header.html" %} et {% include "dsfr/footer.html" %}) et non des tags.

Composants encore non implémentés

  • Barre de recherche (search)
  • Bloc marque (logo)
  • Bouton radio (radio)
  • Bouton radio riche (radio_rich)
  • Case à cocher (checkbox)
  • Citation (quote)
  • Contenu média (media_content)
  • Gestionnaire de consentement (consent)
  • Interrupteur (toggle)
  • Lettre d’information et réseaux sociaux (follow)
  • Liens d'évitement (skiplinks)
  • Modale (modal)
  • Navigation principale (navigation)
  • Onglet (tab)
  • Partage (share)
  • Tag (tag)
